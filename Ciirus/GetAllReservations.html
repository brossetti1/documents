Yes, I am the technical contact for our integration. Non-technical questions would go to Vinny (vinny@ciirus.com).

1. How far back does GetAllReservations() function go? For example will
it retrieve reservations that had an end date of last week?
-This function returns all future reservations. 
2. What is the status of all reservation that are returned? Would they
all be considered book?
-The status could be either a booking or a tentative booking. Since a tentative booking blocks the availability of the property, it can be treated as a regular booking. If the tentative booking expires due to non-payment or any other reason, it frees the availability again automatically. This would be corrected in your system either as you poll at regular intervals or as our system notifies your api of this change.
3. Will cancelled reservations appear on the GerAllReservations() list? 
-If the reservation has been cancelled in the system than it is no longer a reservation. It would only return Bookings which currently occupy space in a property's availability calendar.
